## Display Country ID as Text for Regions Without Text

This code makes it so countries that have no text in online menu will have the country ID as text. For example, North Korea has no text in the original game, but with this code, it will have text, which is "135" (North Korea Country ID). If region display is disabled (no flag, default location), no text will appear. 

Useful if you don't use the Extended Regions mod and want to use my Real-Time Globe Location, Country Flag & Region ID Customizer code. Recommended to use with my Display "Country Unknown" Flag for Unknown/No Flag Regions code to also display unused country unknown flag. 

Each country ID can be found in Extended Regions mod page, you'll see a Region list containing country IDs for each country.

This code is a modification of _tZ's Print BMG IDs code
