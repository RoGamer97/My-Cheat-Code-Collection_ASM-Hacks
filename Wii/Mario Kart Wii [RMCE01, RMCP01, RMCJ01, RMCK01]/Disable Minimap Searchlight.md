## Disable Minimap Searchlight

This code disables the searchlight on the minimap (the yellow arrow pointing to the direction you're going).

<details>
<summary>NTSC-U</summary>

```powerpc
047E1DCC 38600002
```
</details>

<details>
<summary>PAL</summary>

```powerpc
047EB678 38600002
```
</details>

<summary>NTSC-J</summary>

</details>
