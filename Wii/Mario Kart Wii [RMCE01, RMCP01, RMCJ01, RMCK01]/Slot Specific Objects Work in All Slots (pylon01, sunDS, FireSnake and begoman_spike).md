## Slot Specific Objects Work in All Slots (pylon01, sunDS, FireSnake and begoman_spike)

By default, tracks using objects pylon01 (Street cone from Daisy Circuit), sunDS (Angry Sun from DS Desert Hills), FireSnake (Grounded Podoboo from DS Desert Hills), begoman_spike (Topmen from Galaxy Colloseum) only work on Daisy Circuit, Moonview Highway, GBA Shy Guy Beach, DS Desert Hills, and Galaxy Colosseum slots.

This code makes it those objects can work in any slot (Anti Freeze).

