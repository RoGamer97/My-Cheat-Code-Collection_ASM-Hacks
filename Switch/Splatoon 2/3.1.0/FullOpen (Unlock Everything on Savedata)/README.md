## FullOpen (Unlock Everything on Savedata)

FullOpen is a debug feature left in the game that unlocks everything in the savedata and sets your money, levels, ability chunks and anything with quantity to max, and saves it to the savedata. 

Recommended to only use once because it resets certain things like certain NPC message popups as if you opened the game for the first time.

The ASM source code can be found [here](source.s).
